JavaScript Syntax

JavaScript syntax is the set of rules, how JavaScript programs are constructed.

How to create variables:-

var x;
let y;

How to use variables:-

x = 5;
y = 6;
let z = x + y;

JavaScript Values.

The JavaScript syntax defines two types of values:-

  1. Literals are the Fixed values.
  2. Variables are the Variable values .

    1. JavaScript Literals.

      1. JavaScript Numbers.

        2. Number can be written with or without decimals.

      2. JavaScript Strings.

        3. Strings can be written with double or single quotes.

    2. JavaScript Variables

      3. In this example, x is defined as a variable. Then, x is assigned the value of 6

    JavaScript Operators.

    JavaScript uses arithmetic operators to compute values (just like algebra).

    ADD

    SUB

    MULTIPLY

    DIVIDE

    1. Assignment Operator.

    In JavaScript the (=) sign operator is used to assign values to variables.

    JavaScript Expressions.

    JavaScript Expressions is used to compute the given values.

    This JavaScript Expressions can also contain variable value.

    Expressions compute to values.

    The JavaScript Keywords are used to identify actions to be performed.

    1. JavaScript 'let' Keyword

      2. The 'let' Keyword tells the browser to create variable.
